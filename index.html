¿Por qué usar Brackets?

Brackets es un editor de texto ligero pero poderoso. Hemos agregado herramientas visuales dentro del editor para que tengas la dosis de ayuda correcta cuando quieras sin interrumpir tu proceso creativo.

Disfrutarás escribiendo código en Brackets.

Hecho con y JavaScript

Brackets es un proyecto open-source, soportado por una comunidad activa y apasionadas. ¡Está hecho por otros desarrolladores web como tú!

Características

1

Editores por línea

En lugar de cambiar entre ventadas de archivos, Brackets permite abrir una ventana para el código que más te importa.

2

Vista previa

Obtén una conexión en tiempo real con tu navegador. Realiza cambios en CSS y HTML y verás instantáneamente los cambios en pantalla.

3

Soporta preprocesamiento

Trabajo con preprocesamiento de una nueva forma. Sabemos lo importante que es el proprocesamiento para un flujo de trabajo.
